# Escaping

默认情况下，块级 (block) 的代码块将会被自动包裹在v-pre中。如果你想要在内联 (inline) 的代码块或者普通文本中显示原始的大括号，或者一些 Vue 特定的语法，你需要使用自定义容器 v-pre 来包裹.
